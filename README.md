# 💰 Cryptocurrency Portfolio Tracker

A web-based Cryptocurrency Portfolio Tracker built using Python (Flask), HTML, and CSS. This application allows users to manage their crypto investments and track real-time prices through an interactive dashboard.

---

## 🚀 Features

* ➕ Add cryptocurrencies (Bitcoin, Ethereum, etc.)
* ✏️ Edit existing portfolio entries
* 🗑 Delete cryptocurrencies
* 📊 Dashboard displaying:

  * Total Investment
  * Current Portfolio Value
  * Net Profit/Loss
  * Best Performing Coin
  * Worst Performing Coin
* 🌐 Real-time price updates using CoinGecko API
* 💾 Data stored using JSON (persistent storage)
* 🎨 Clean and modern user interface

---

## 🛠️ Tech Stack

* **Backend:** Python (Flask)
* **Frontend:** HTML, CSS
* **API:** CoinGecko API
* **Storage:** JSON file

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/pavanipoluri/CryptoCurrency-Portfolio-Tracker.git
```

2. Navigate to the project folder:

```
cd CryptoCurrency-Portfolio-Tracker
```

3. Install dependencies:

```
pip install flask requests
```

4. Run the application:

```
python app.py
```

5. Open in browser:

```
http://127.0.0.1:5000
```
